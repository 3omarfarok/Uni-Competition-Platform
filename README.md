# University Competition Platform 🏆

This project is a **semi-professional web platform** built for a **university competition**, developed as part of a **Pearson BTEC Diploma**.

The system allows participants to register and apply for the competition either **individually or as part of a team**, while **admins manage and control participants’ scores**.

The project was built mainly using **PHP**, focusing on backend logic and system flow.

Uploaded for **archiving and learning documentation purposes**.

---

## 🧰 Technologies Used

- PHP  
- HTML  
- CSS  
- Bootstrap  
- Cookies & Sessions  
- MySQL  

---

## 📄 Project Overview

- User registration and login system  
- Individual and group participation  
- Admin roles for managing participants and points  
- Session-based authentication
